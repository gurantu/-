# 📊 デイトレ記録ダッシュボード

証券会社の約定履歴CSVファイルを使って、デイトレードの成績を可視化・分析するWebアプリケーションです。

## ✨ 特徴

- 🌐 **完全オフライン動作** - サーバー不要、ブラウザだけで動作
- 💾 **LocalStorage保存** - データはあなたのブラウザにのみ保存
- 📈 **多角的な分析** - 手法別、期間別、特殊条件での統計表示
- 📱 **SNS投稿サポート** - X（Twitter）用のテキスト自動生成
- 🔒 **プライバシー保護** - データは外部に送信されません

## 🚀 使い方

### 1. CSVファイルの準備

証券会社からダウンロードした約定履歴CSVファイルを用意してください。
※「sample.csv」はデモ用データです

**必要な列:**
- 日付（例: `2024/01/15`）
- 銘柄名（例: `トヨタ自動車`）
- 損益（例: `+5000` または `-3000`）

**対応CSV例:**
```csv
日付,銘柄名,損益
2024/01/15,トヨタ自動車,+5000
2024/01/15,ソニーグループ,-2000
2024/01/16,三菱UFJ,+8000
```

### 2. データのアップロード

1. ブラウザで `index.html` を開く
2. 「📂 CSVファイルを選択」ボタンをクリック
3. 用意したCSVファイルを選択
4. 自動的にデータが読み込まれ、統計が表示されます

### 3. 詳細情報の追加

各トレードに以下の情報を追加できます：

- **手法**: ドロップダウンから選択（押し目、逆張り、ブレイク、その他）
- **下ヒゲの有無**: チェックボックスで記録
- **検証損益**: 「もしこうしていたら」のシミュレーション値

### 4. 統計の確認

**4つの統計タブ:**
1. **実トレード手法別** - 実際の取引を手法ごとに集計
2. **実トレード特殊統計** - 下ヒゲの有無での成績比較
3. **検証手法別** - 検証損益を手法ごとに集計
4. **検証特殊統計** - 検証損益での下ヒゲ比較

**期間フィルタ:**
- 全期間
- 今月
- 今週

### 5. X投稿用テキスト生成

「📱 X投稿用テキスト生成」ボタンをクリックすると、以下の形式でテキストが生成されます：

```
【今週のトレード結果】

2024/01/15
・トヨタ自動車(押し目) +5,000円
・ソニーグループ(逆張り) -2,000円
→ 日次: +3,000円

今週合計: +3,000円
勝率: 1/1日 (100.0%)

#デイトレ #株式投資
```

## 📋 主な機能

| 機能 | 説明 |
|------|------|
| CSV読み込み | 証券会社の約定履歴をアップロード |
| 統計表示 | 総損益、勝率、平均損益をリアルタイム計算 |
| 手法管理 | 各トレードに手法を割り当て |
| 検証機能 | 実トレードとは別に検証損益を記録 |
| 期間フィルタ | 全期間/今月/今週で絞り込み |
| JSON出力 | データをJSON形式でエクスポート |
| データ削除 | 全データを一括削除 |

## 🔧 カスタマイズ

### 手法の変更

コード内の `methods` 配列を編集することで、手法の選択肢をカスタマイズできます：

```javascript
const methods = ['押し目', '逆張り', 'ブレイク', 'その他'];
// 例: const methods = ['順張り', '逆張り', 'スキャルピング', 'スイング'];
```

### ハッシュタグの変更

X投稿用テキストのハッシュタグは `generatePost()` 関数内で変更できます：

```javascript
postText += '#デイトレ #株式投資';
// 例: postText += '#株 #投資記録 #トレード日記';
```

## ⚠️ 注意事項

- **データの保管**: ブラウザのLocalStorageに保存されるため、ブラウザのデータ削除やキャッシュクリアでデータが失われます
- **バックアップ推奨**: 定期的に「💾 JSONダウンロード」でバックアップを取ることを推奨します
- **ブラウザ依存**: 同じブラウザでのみデータにアクセスできます
- **セキュリティ**: データは外部に送信されませんが、共有PCでの使用には注意してください
